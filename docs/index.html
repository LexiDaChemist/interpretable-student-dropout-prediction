Student Outcome Prediction

Predicts whether a student is likely to Dropout, stay Enrolled, or Graduate.

View Code on GitHub

What this project does

This project uses historical student data to predict academic outcomes. It also explains why the model makes each prediction using interpretable machine learning.

Predicted outcomes

  • Dropout
  • Enrolled
  • Graduate

Key features used

  • Age at enrollment
  • Tuition payment status
  • Curricular units approved
  • Scholarship status
  • Enrollment characteristics
