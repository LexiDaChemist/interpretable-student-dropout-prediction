Student Outcome Prediction
Overview Results Interpretability View Code on GitHub

Predicts whether a student is likely to Dropout, stay Enrolled, or Graduate.

A logistic regression model trained on historical student records. Includes evaluation + interpretable explanations of the strongest factors driving each predicted outcome.

Project Overview

This project uses student demographics, enrollment status, and academic progress features to predict outcomes. It also produces interpretability visuals showing which features push the model toward each class.

Predicted outcomes

  • Dropout
  • Enrolled
  • Graduate

Key features used

  • Age at enrollment
  • Tuition fees up to date
  • Scholarship holder
  • Debtor
  • Unemployment rate
  • Academic unit approvals / grades
